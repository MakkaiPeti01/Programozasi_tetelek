# Programozási tetelek
## Elõkészítés
- Tíz elemû tömböt létrehozása
- Tömb elemei:
      
     - 12, 7, 4, 45, 99, 1, 8, 95, 50
    - A tömb az összes metúdusban látható legyen.
    - TömbKiiras() metódus létrehozása.
    - Nem ad vissza értéket, nincs paramétere.
    - A tömb elemeit vesszõvel elválasztva egy sorban a console irja ki.
## Összegzés
- Összegzés metódus létrehozása
- Összegzés tétele a tömbön
- Hány 3al osztható szám van a tömbben
- Kiírás

## Eldöntés
- Metódus létrehozása
- Van-e a tömbben "69"?
- Kiírás ("Van/Nincs")

## Kiválasztás
- Metódus létrehozása
- Hányadik az "50"?
- Kiírás az indexét.
